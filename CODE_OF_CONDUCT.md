# 行为准则 · Code of Conduct

```
>_ HOUSE RULES
```

Styledex 想做一个友善、专业、对新人友好的社区。参与本项目（Issue、PR、Discussion）即表示你同意：

- **友善、就事论事。** 设计审美可以激烈讨论，但对人保持尊重。
- **欢迎新人。** 第一次提 PR 写得不完美没关系，我们一起改。
- **不歧视、不骚扰。** 不容忍任何针对性别、种族、地域、性取向、经验水平等的攻击。
- **保持建设性。** 批评请给出可操作的建议。

不当行为可通过 Issue 或仓库维护者邮箱反馈，维护者会处理，包括必要时移除评论、提交或参与资格。

本准则精神参考 [Contributor Covenant](https://www.contributor-covenant.org/)。

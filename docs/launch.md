## 社区长文（论坛 / Show HN / 即刻 / V2EX 等）

**标题**：Styledex — 一个开源的「设计风格图鉴」，帮你在说不出名字之前先找到想要的风格

正文：

> **痛点**：独立开发者、初级设计师常常能分辨「好看」，却说不清自己想要什么风格，更不知道怎么落地。现有工具要么要求你已经知道要找什么（Mobbin），要么只是好看图片的海洋（Pinterest）。
>
> **Styledex 的做法**：把命名风格当入口（先用眼睛认出），把可操作的风格配方当价值（配色 / 排版 / prompt 可直接拿走）。
>
> 几个我自己比较满意的点：
> - **配色 + 排版 双主轴**：排版（对齐/字阶/密度/网格/字形）是一等检索维度，可以「按版式找风格」。
> - **可交互的排版天性**：点选排版轴，缩略图和放大详图实时重排。
> - **跨品类样本**：同一风格在网页/App/海报/包装下分别长什么样，每类 6 种版式。
> - **核心资产是风格配方而非图片库**：样本全是 CSS 程序生成，零图片版权包袱，也因此特别适合社区共建。
>
> 单文件 HTML、零依赖、MIT 协议。最想要的贡献就是**更多风格**——加一个风格基本就是加一段结构化数据。
>
> Demo + 代码：github.com/underdogYnino/Styledex
>
> 欢迎拍砖 / 提风格 / 提 PR。
